# Images pour le système de pointage

## background-pointage.jpg

Cette image sera utilisée comme fond d'écran pour l'interface de pointage (QuickPointage).

**Spécifications recommandées :**
- Format : JPG ou PNG
- Dimensions : 1920x1080 pixels (Full HD) ou plus
- Poids : < 500 KB (optimisé pour le mobile)
- Style : Image professionnelle liée à l'entreprise ou au lieu de travail

**Pour ajouter votre image :**
1. Placez votre image dans ce dossier
2. Renommez-la : `background-pointage.jpg`
3. L'image sera automatiquement utilisée comme fond d'écran

**Note :** Si l'image n'est pas présente, un fond bleu uni sera affiché par défaut.
